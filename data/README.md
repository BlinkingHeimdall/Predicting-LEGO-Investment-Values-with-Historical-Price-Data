# Data Folder

This project includes both raw and processed data to ensure reproducibility for WGU acedemic evaluation.

## Raw Data

- `data/raw/Kaggle_LEGO_DATASET.csv`
- Source: https://www.kaggle.com/datasets/alexracape/lego-sets-and-prices-over-time
- License: MIT


## Processed Data
- `data/processed/lego_model_ready.csv`
- Generated by running:
  `python -m src.make_dataset`
